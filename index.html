Preguntas HTML

¿Qué es HTML y cuál es su función principal en el desarrollo web?
HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Su función principal es estructurar el contenido de una página mediante etiquetas, permitiendo la organización de textos, imágenes, enlaces y otros elementos.
¿Qué son las etiquetas HTML y cómo se utilizan?
Las etiquetas HTML son los elementos que estructuran el contenido de una página web. Se utilizan encerrando el contenido entre etiquetas de apertura y cierre, como <p>Este es un párrafo</p>. Algunas etiquetas no requieren cierre, como <img>.
Proporciona ejemplos de algunas de las etiquetas HTML más comunes
  • <h1> a <h6>: Encabezados
  • <p>: Párrafos
  • <a href="">: Enlaces
  • <img src="">: Imágenes
  • <ul> y <ol>: Listas

Preguntas CSS

¿Qué es CSS y cómo se utiliza para dar estilo a una página web?
CSS (Cascading Style Sheets) es un lenguaje de estilos que se usa para dar diseño y formato a una página web mediante reglas que afectan la apariencia de los elementos HTML.
¿Cuáles son las propiedades más importantes de CSS?
Las propiedades más importantes de CSS incluyen color, font-size, margin, padding, display, position, width y height.
Explica el modelo de caja (box model) en CSS.
El modelo de caja en CSS representa cómo los elementos HTML se estructuran en rectángulos con content, padding, border y margin.
¿Qué es el diseño responsivo y cómo se logra con CSS?
El diseño responsivo permite que una web se adapte a diferentes tamaños de pantalla y se logra con media queries, flexbox, grid y unidades relativas como % y rem.

Preguntas sobre JavaScript

¿Qué es JavaScript y cómo se integra en el desarrollo web?
JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web. Se integra en el desarrollo web mediante scripts que pueden estar dentro del HTML (en etiquetas <script>) o en archivos externos.
Explica cómo JavaScript interactúa con el HTML y CSS a través del DOM (Document Object Model).
JavaScript interactúa con HTML y CSS a través del DOM (Document Object Model), que representa la estructura de la página web como un árbol de nodos. Con JavaScript, se pueden modificar los elementos, cambiar estilos, agregar o eliminar contenido dinámicamente.
Proporciona ejemplos de cómo JavaScript puede hacer que una página web sea interactiva.
Algunos ejemplos incluyen:
  • Mostrar u ocultar elementos con eventos como clics.
  • Validar formularios antes de enviarlos.
  • Cambiar el contenido de la página sin recargarla (AJAX).
  • Crear efectos visuales como animaciones y transiciones.

Preguntas sobre Git y GitHub

¿Qué es Git y por qué es importante para el desarrollo web?
Git es un sistema de control de versiones distribuido que permite gestionar cambios en el código fuente de un proyecto. Es importante en el desarrollo web porque facilita el seguimiento de modificaciones, permite trabajar en equipo sin conflictos y mantiene un historial de versiones.
Explica cómo Git se usa para el control de versiones en proyectos de desarrollo web.
Git permite gestionar versiones de un proyecto mediante comandos como git init, git add, git commit y git push. Con Git, los desarrolladores pueden trabajar en ramas separadas, fusionar cambios y revertir errores sin afectar la versión principal del código.
¿Qué es GitHub y cómo facilita la colaboración en proyectos web?
GitHub es una plataforma basada en la nube que permite alojar repositorios Git y facilita la colaboración entre desarrolladores. Ofrece herramientas como pull requests, issues y revisión de código, lo que permite a los equipos trabajar de manera eficiente en proyectos web compartidos.
